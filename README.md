# 🌍 ESG Site Performance Dashboard

An interactive ESG (Environmental, Social, Governance) dashboard designed to monitor and analyze site-level performance across key sustainability and operational metrics.

---

## 📊 Project Overview

This project provides a comprehensive view of ESG performance across multiple sites, focusing on:

- 🌱 Environmental impact (Emissions)
- 👩‍💼 Workforce diversity (Female workforce %)
- ⚠️ Safety performance (LTIR - Lost Time Injury Rate)
- 📚 Employee development (Training hours)

The dashboard enables stakeholders to track KPIs, identify trends, and make data-driven decisions.

---

## 🚀 Tools & Technologies

- **Python** → Data cleaning, transformation, preprocessing  
- **SQL** → Data extraction, querying, and aggregation  
- **Tableau** → Data visualization and dashboard creation  

---

## 📈 Key Features

- **Total Emissions Tracking**
- **Average Female Workforce %**
- **LTIR Monitoring (Safety KPI)**
- **Total Recordable Incidents**
- **Site-wise breakdown of all metrics**
- **Interactive filters** (Business Unit, Site Type, State)
- **Comparative analysis across sites**

---

## 📌 Dashboard Insights

- Identify high-emission sites and monitor reduction efforts  
- Analyze workforce diversity trends across locations  
- Track safety performance using LTIR  
- Compare training hours to evaluate employee development
