# ESG Site Performance Dashboard

An interactive ESG analytics project built using **Python, SQL, and Tableau** to track and evaluate site-level performance across key sustainability and workforce metrics.

## Project Overview

This project analyzes ESG performance across multiple sites by combining environmental, safety, and social indicators into a single dashboard. The objective is to clean, transform, and visualize the data in a way that supports monitoring, comparison, and business decision-making.

The dashboard focuses on metrics such as:
- Emissions
- Safety performance (LTIR)
- Workforce diversity
- Training hours across sites

## Tools Used

- **Python** for data cleaning, preprocessing, and analysis
- **SQL** for querying, joining, and transforming structured datasets
- **Tableau** for building the final interactive dashboard

## Project Workflow

### 1. Data Collection
The project uses multiple CSV files containing site-level ESG information, including emissions, HR, and site reference data.

### 2. Data Cleaning and Preparation
Using Python, the raw datasets were reviewed, cleaned, and standardized. This included:
- handling missing values
- checking data consistency
- merging relevant datasets
- preparing summary outputs for dashboarding

### 3. Data Querying with SQL
SQL was used to:
- join datasets
- aggregate site-level KPIs
- create structured outputs for reporting and analysis

### 4. Dashboard Development
The cleaned and transformed data was used in Tableau to create an interactive dashboard for comparing performance across sites and identifying trends.

## Key Dashboard Metrics

The dashboard includes analysis of:
- total emissions by site
- Lost Time Injury Rate (LTIR)
- workforce diversity indicators
- employee training metrics
- site-level ESG comparisons
